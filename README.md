
# vue3 + vite + pinia + Qiankun

# install
1. clone

   ```bash
   git clone https://github.com/xiamiLi/combinePro.git
   ```

2. Depend on the installation

   ```bash
   cd combinePro
   npm install
   ```

3. start

> master application 
   ```bash
   cd combinePro
   npm run dev
   ```
> micro application 
 ```bash
   cd sub-vue
   npm run start
   ```
